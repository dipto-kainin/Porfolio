# Product

## Register

brand

## Users
Recruiters, hiring managers, and developers scanning the portfolio to assess the owner's skills in frontend development, system design, and product engineering.

## Product Purpose
A mock OS desktop portfolio simulating a real desktop environment to showcase projects, experience, skills, and contact options in an engaging, interactive format.

## Brand Personality
- Technical confidence (clean, dark tech)
- High polish (smooth transitions, glassmorphism, precise layouts)
- Restrained playfulness (retro/modern desktop aesthetic)

## Anti-references
- Cluttered layouts with overly bright, distracting animations
- AI-purple gradient mesh marketing sites
- Static text-only portfolios

## Design Principles
- **OS Realism**: Interactive elements should behave like a desktop OS (dragging, minimizing, snapping, focusing).
- **Material Polish**: Utilize glassmorphic styling, precise card corners, and subtle light-themed drop shadows to give depth.
- **Developer Focus**: Maintain readable typography and high contrast for code stacks and technical details.

## Accessibility & Inclusion
- Contrast ratio ≥ 4.5:1 for readability.
- Honor user reduced-motion preferences.
- Accessible interactive button labels and keyboard navigation.
